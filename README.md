## Sobre

Projeto realizado no Curso de Dev Fullstack de Fullture, na criação de uma loja de bonecos funcional e atrativa, com desing responsivo, página de catálogo e página de contatos.

## Pré-requisitos

Para instalar o software, você precisará das seguintes ferramentas:

- VS Code
- Git
- Github

## Status do Projeto

> Status do Projeto: Concluído :heavy_check_mark:

## Como baixar o projeto

```bash
# Opção 1: Clonar o repositório
$ git clone https://github.com/rodrusantu-dev/LojaSuperHero

# Opção 2: Fazer o download do Projeto
* Acessar via VS Code no seu computador.

```
